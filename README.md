# Meow 🐱

> 这家伙不仅能摸鱼，现在还能帮你过日子。除了不掉毛，日历、记账、闹钟、网页涂鸦等样样精通。

一款功能丰富的 Chrome 扩展（Manifest V3），集成了日历待办、财务记账、习惯打卡、网页涂鸦、工具箱等近 **30 个功能模块**，是你的浏览器全能助手。

## 🚀 安装

### 从源码加载（开发者模式）

1. 打开 Chrome 浏览器，进入 `chrome://extensions`
2. 开启右上角 **「开发者模式」**
3. 点击 **「加载已解压的扩展程序」**
4. 选择本项目根目录 `Meow/` 文件夹

无需安装依赖，零构建流程，加载即用。

## ⌨️ 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Alt+Q` | 打开/关闭主弹出面板 |
| `Alt+W` | 切换网页备注模式 |
| `Alt+C` | 打开侧边栏工具集 |
| `Alt+R` | 打开大日历视图 |
| `Alt+S` | 保存当前页面到稍后阅读 |
| `Esc` | 关闭面板 / 退出备注模式 |
| `Ctrl+Z` | 备注模式撤销 |
| `Alt+F1` | 备注模式切换画笔工具 |

> 所有快捷键可在 `chrome://extensions/shortcuts` 中自定义。

## 🧩 技术栈

| 类别 | 技术 |
|------|------|
| **运行时** | Chrome Extension Manifest V3 |
| **语言** | 原生 JavaScript (ES6+)，无 UI 框架 |
| **存储** | `chrome.storage.sync`（同步数据）+ `chrome.storage.local`（大容量） |
| **音频** | Web Audio API（Offscreen Document 播放铃声） |
| **图标** | Material Icons（Google Fonts CDN） |
| **第三方库** | flatpickr（日期选择器）、lunar.min.js（农历） |

> 纯原生开发，无打包工具、无构建步骤、无 runtime 依赖。

## 💾 数据安全

- **存储方式**: 所有数据存储在本地 `chrome.storage` 中，不上传任何第三方服务器
- **WebDAV 备份**: 支持配置自建 WebDAV 服务器进行云端备份（手动/自动）
- **完整导出**: 支持一键导出全部数据为 JSON 文件
- **财务导出**: 支持将记账数据导出为 CSV
